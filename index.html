¿Qué es HTML5?

Publicado el 10/07/2018 por UMG-Jalapa

Durante los últimos años, el estándar HTML5 se ha estado afianzando en muchas páginas web. Sin embargo, ¿qué queremos decir con esto? ¿Cuál es la diferencia crucial entre el HTML "común", por así decirlo, y este nuevo estándar? Hoy hablaremos sobre esto y responderemos a la pregunta de qué es HTML5.

HTML5 es un lenguaje más semántico. En otras palabras, se le da un significado a las diferentes partes de la páginas Web, de forma que el navegador puede distinguir la parte que forma parte de la cabecera gracias a que hay una etiqueta que dice "Cabecera", bueno lo dice en inglés: <header>. ¿Para qué sirve esto? Ayuda a los buscadores como Google a entender tu página Web, algo que se espera sea positivo de cara al SEO, y también sirve para que los navegadores muestren de forma inteligente diferentes versiones de la página Web sin tener que usar plug-ins para dispositivos específicos.

Introducción a CSS3

Publicado el 10/07/2018 por UMG-Jalapa

El nombre hojas de estilo en cascada viene del inglés Cascading Style Sheets, del que toma sus siglas. CSS es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML o XML (y por extensión en XHTML). El W3C (World Wide Web Consortium) es el encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.

¿Para qué sirve el CSS3?

El CSS sirve para definir la estética de un sitio web en un documento externo y eso mismo permite que modificando ese documento (la hoja CSS) podamos cambiar la apariencia de todo un sitio web, el mismo sitio web puede variar totalmente de estética cambiando solo la CSS, sin tocar para nada los documentos HTML o .js o asp que lo hacen funcionar.

¿Qué es y para qué sirve JavaScript?

JavaScript (JS) es un lenguaje ligero e interpretado, orientado a objetos con funciones de primera clase, más conocido como el lenguaje de script para páginas web, pero también usado en muchos entornos sin navegador, tales como node.js.

Es el lenguaje de programación más conocido para el desarrollo web, permitiendo la interacción dinámica con el contenido del sitio, como la validación de formularios, creación de juegos, animaciones, y mucho más.

Acerca de Nosotros

Somos un grupo de entusiastas y profesionales del desarrollo web dedicados a promover el uso y la adopción de HTML5 en la industria. Desde que HTML5 fue presentado, hemos trabajado arduamente para educar a la comunidad de desarrolladores sobre sus beneficios y capacidades.

A través de nuestros recursos educativos, talleres y noticias, buscamos ser una fuente de conocimiento confiable y actualizada sobre todo lo relacionado con HTML5.

Únete a nuestra comunidad y mantente al tanto de las últimas tendencias y avances en HTML5.
